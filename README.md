# NewsExplorer Backend
Este es un servicio que te permite encontrar noticias y guardarlas en tu cuenta.

## Como usar
Instalar las dependencias:
```
$ npm install
```
Y después:
```
$ npm run dev
```

## Tecnologias Utilizadas
* Node js
* Express js
* MongoDB
